# Conversor de imagem

Fiz esse projeto pra resolver o problema de converter fotos .heic do iphone pra jpg ou png direto no navegador, sem ter que ficar subindo foto em site cheio de vírus.

## Tecnologias que usei:

* Nextjs
* Tailwind
* jszip (pro zip)
* heic2any (pra ler o formato da apple)

## Como rodar o projeto:

Primeiro instala as dependencias:
`npm install`

Depois roda o servidor:
`npm run dev`
